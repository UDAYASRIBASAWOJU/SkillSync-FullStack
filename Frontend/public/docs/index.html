Layered Architecture Flow

Layer 1

Browser / ReactRedux + Axios interceptors

HTTPS (TLS encrypted)
Layer 2

Nginx (entry gate)

Force HTTPS SSL termination
Security headers MSTS - XSS - X-Frame
Reverse proxy → 127.0.0.1:8080 Adds X-Forwarded-For - X-Forwarded-Proto
Internal HTTP (decrypted)
Layer 3

API Gateway (Spring Cloud + Netty)

Rate limiting 10/5/100 req/min
JWT validation Sig + expiry check
Header spoofing Strip & re-inject
Eureka service discovery lb://auth-service, user-service
Block internal routes /api/auth/internal/** → 403
Trusted X-User-* headers injected
Layer 4

Apache Tomcat (auth-service)

NioEndpoint Thread pool assign
Http11Processor + CoyoteAdapter TCP bytes → HttpServletRequest
DelegatingFilterProxy → springSecurityFilterChain bridge between Tomcat → Spring Security
Layer 5

Spring Security filter chain

SecurityContextPersistenceFilter load + clear ThreadLocal
HeaderWriterFilter + CorsFilter XSS - clickjack - CORS headers
CsrfFilter (disabled) JWT = stateless, no session
LogoutFilter clears context + /logout
⭐ JwtAuthenticationFilter (custom) Extract → verify sig → check expiry → set SecurityContextHolder
AnonymousAuthenticationFilter assign guest if no JWT
ExceptionTranslationFilter AuthException→401 - AccessDenied→403
⭐ FilterSecurityInterceptor Role check - @PreAuthorize - AffirmativeBased voting
Layer 6

Spring MVC / DispatcherServlet

AuthController.login() receives LoginRequest DTO
AuthService.login() delegates to AuthManager
ProviderManager loops auth providers
DaoAuthProvider UserDetailsService + BCrypt
@RestControllerAdvice handles business exceptions (Filter exceptions → ExceptionTranslationFilter, not ControllerAdvice)
Layer 7

Security internals

ThreadLocal / SecurityContext per-thread user identity - cleared after request
BCrypt password hashing slow + salted → brute-force safe - password cleared post-auth
JWT (HS256) header.payload.sig - stateless + signed
Token storage strategy Access → Redux/localStorage
Refresh → HttpOnly cookie
Layer 8

Data + cache layer

PostgreSQL users - refresh tokens - roles
refresh token rotation (DB)
Redis (caching) user profile cache + eviction
JWT blacklist (planned TTL)
Layer 9

Response flow back

Access token → body Refresh → HttpOnly cookie (Secure - SameSite=None)
Nginx re-encrypts HTTPS back to browser
Thread cleaned (ThreadLocal)
Layer 10

Browser state update

Redux authSlice: isAuthenticated=true ProtectedRoute unlocks → dashboard
Self-healing: 401 → /refresh → retry
